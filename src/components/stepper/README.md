### API

#### Stepper Props
属性    | 说明 | 类型 | 默认值
---------|------|--------|----
default-value |默认值| Number|-
step|每次改变步数，可以为小数|Number|`1`
min|最小值|Number|`-Infinity`
max|最大值|Number|`Infinity`
disabled|禁用| Boolean|`false`
read-only|只读| Boolean|`false`

#### Stepper Events

##### @change()
值发生变化事件
